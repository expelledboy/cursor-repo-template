# System Domain

## Purpose
Define the scope and boundaries of the docs system domain.

## Scope
- Documentation system rules and governance.
- Tooling that renders or validates the docs graph.
- Domain, intent, task, and contract definitions.

## Boundaries
- Does not include product or application docs.
- Does not define authority for other domains.
