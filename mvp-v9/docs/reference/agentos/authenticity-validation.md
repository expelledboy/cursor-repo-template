# Authenticity Validation (Reference)

## Core Authenticity Requirements

### Intelligence Separation
AgentOS v9 maintains strict separation between deterministic evidence collection (E layer) and intelligent analysis (O layer).

**Scripts SHALL:**
- Provide structured evidence only
- Execute deterministic operations
- Remain purely functional and predictable
- Never make assumptions or perform intelligence work

**AI Agents SHALL:**
- Perform all analysis and interpretation
- Make decisions and recommendations
- Generate insights and categorizations
- Handle complex reasoning and pattern recognition

### Self-Awareness Validation
**MUST** identify docs-implementation gaps through evidence collection
**MUST** maintain authoritative hierarchy consciousness
**MUST** use evidence for genuine self-improvement
**MUST NOT** perform blind automation or assumption-making

### DOE Framework Compliance
**MUST** operate within defined layer responsibilities
**MUST** maintain authoritative precedence hierarchy
**MUST** provide evidence for orchestration layer analysis
**MUST** respect governance flow: framework → layer specs → component specs → implementations
